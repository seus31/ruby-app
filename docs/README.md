## アプリ概要

個人・チーム向けのブログプラットフォーム。ユーザーは記事を投稿し、カテゴリやタグで整理できる。読者はコメントを投稿し、記事にいいねを付けることができる。

### 主要機能

- **ユーザー管理**: 会員登録・ログイン（JWT認証）・プロフィール編集
- **記事管理**: 記事の作成・編集・削除・公開/下書き切り替え
- **カテゴリ**: 記事をカテゴリで分類（多対多）
- **タグ**: 記事にタグを付与（多対多）
- **コメント**: 記事へのコメント投稿・返信（ネスト対応）
- **いいね**: 記事へのいいね機能
- **検索**: タイトル・本文・タグによる全文検索

## 技術スタック

- **Frontend**: Next.js 15.1.6 (App Router), React 19, TypeScript, Bootstrap 5
- **Backend**: Ruby on Rails 7.1.3 (API Mode), MySQL 8.0, JWT認証
- **Infrastructure**: Docker Compose
